# 📊 Sales Data Dashboard

## Overview

The **Sales Data Dashboard** is a beginner-friendly HTML project created to display sales information in a simple and organized format. It presents sales data using tables and basic HTML elements without using CSS, JavaScript, or images.

This project is suitable for beginners who are learning HTML and want to build a basic dashboard for their GitHub portfolio.

## Features

* Displays total sales, orders, and customers
* Monthly sales report in a table
* Top-selling products table
* Sales insights section
* Simple and clean HTML structure
* No images used
* Easy to understand and modify

## Technologies Used

* HTML5

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click **index.html** or open it in any web browser.
